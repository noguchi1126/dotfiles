MongoDB\BSON\ObjectID::__toString

(mongodb >=1.0.0)

MongoDB\BSON\ObjectID::__toString — Returns the hexidecimal representation of this ObjectID

説明

final public string MongoDB\BSON\ObjectID::__toString ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns the hexidecimal representation of this ObjectID.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。

例

例1 MongoDB\BSON\ObjectID::__toString() example

<?php

var_dump((string) new MongoDB\BSON\ObjectId());
var_dump((string) new MongoDB\BSON\ObjectId('000000000000000000000001'));

?>

上の例の出力は、 たとえば以下のようになります。

string(24) "56731b49da14d8747d701211"
string(24) "000000000000000000000001"
