session_abort

(PHP 5 >= 5.6.0, PHP 7)

session_abort — session 配列の変更を破棄してセッションを終了します

説明

void session_abort ( void )

session_abort() は、データを保存せずにセッションを終了します。 したがって、セッション・データの元の値が保持されます。

返り値

値を返しません。

参考

  • $_SESSION
  • session.auto_start 構成ディレクティブ
  • session_start() - 新しいセッションを開始、あるいは既存のセッションを再開する
  • session_reset() - session 配列を元の値で再初期化します
  • session_commit() - session_write_close のエイリアス
