MongoDB\BSON\UTCDateTime::serialize

(mongodb >=1.2.0)

MongoDB\BSON\UTCDateTime::serialize — Serialize a UTCDateTime

説明

final public string MongoDB\BSON\UTCDateTime::serialize ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns the serialized representation of the MongoDB\BSON\UTCDateTime.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。
