SVMModel::getLabels

(PECL svm >= 0.1.5)

SVMModel::getLabels — モデルが訓練されたラベルを取得

説明

public array SVMModel::getLabels ( void )

モデルが訓練されたラベルの配列を返します。 回帰および 1 つのクラスのモデルでは、空の配列が返されます。

パラメータ

この関数にはパラメータはありません。

返り値

ラベルの配列を返します

参考

  • SVMModel::getNrClass() - モデルが訓練されたクラスの個数を返す
