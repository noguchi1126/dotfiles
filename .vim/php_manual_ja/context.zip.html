Zip context options

Zip context options — Zip context option listing

説明

Zip context options are available for zip wrappers.

オプション

password

Used to specify password used for encrypted archive.

変更履歴

バージョン 説明
PHP 7.2.0, PECL zip 1.14.0 Added password.

例

例1 Basic password usage example

<?php
// Read encrypted archie
$opts = array(
    'zip' => array(
        'password' => 'secret',
    ),
);
// create the context...
$context = stream_context_create($opts);

// ...and use it to fetch the data
echo file_get_contents('zip://test.zip#test.txt", false, $ctx);

?>
