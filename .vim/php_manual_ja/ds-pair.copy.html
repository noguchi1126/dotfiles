Ds\Pair::copy

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Ds\Pair::copy — Returns a shallow copy of the pair.

説明

public Ds\Pair Ds\Pair::copy ( void )

Returns a shallow copy of the pair.

パラメータ

この関数にはパラメータはありません。

返り値

Returns a shallow copy of the pair.

例

例1 Ds\Pair::copy() example

<?php
$a = new \Ds\Pair("a", 1);
$b = $a->copy();

$a->key = "x";

print_r($a);
print_r($b);
?>

上の例の出力は、 たとえば以下のようになります。

Ds\Pair Object
(
    [key] => x
    [value] => 1
)
Ds\Pair Object
(
    [key] => a
    [value] => 1
)
