Throwable::getTrace

(PHP 7)

Throwable::getTrace — スタックトレースを取得する

説明

abstract public array Throwable::getTrace ( void )

スタックトレースの配列を返します。

パラメータ

この関数にはパラメータはありません。

返り値

スタックトレースの配列を返します。その書式は debug_backtrace() と同じです。
