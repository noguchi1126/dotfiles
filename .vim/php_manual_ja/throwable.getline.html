Throwable::getLine

(PHP 7)

Throwable::getLine — オブジェクトのインスタンスが作られた行番号を取得する

説明

abstract public int Throwable::getLine ( void )

スローされたオブジェクトのインスタンスが作られた行番号を返します。

パラメータ

この関数にはパラメータはありません。

返り値

スローされたオブジェクトのインスタンスが作られた行番号を返します。
